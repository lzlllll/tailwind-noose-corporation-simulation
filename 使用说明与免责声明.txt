# 使用说明与免责声明

欢迎使用本项目。

本项目为个人开发的实验性 AI 模拟经营游戏，目前仍处于 Pre-Alpha（早期测试）阶段。

游戏中的部分文本、事件、新闻及世界内容由 AI 自动生成，可能存在：

* 错误、不准确或不符合现实的内容
* 虚构政治、经济或社会观点
* 不完整、逻辑异常或不符合常识的描述
* 偶发的不当、偏激或令人不适的生成结果

上述内容均为 AI 自动生成结果，仅用于虚构世界模拟与玩法体验，不代表开发者本人立场、观点或价值倾向。

# 用户行为规范

请勿利用本项目生成、传播或讨论任何违反中华人民共和国法律法规的内容，包括但不限于：

* 煽动违法犯罪
* 仇恨、歧视、极端主义内容
* 色情、暴力、恐怖主义相关内容
* 非法政治宣传
* 恶意骚扰、人身攻击
* 侵犯他人隐私或合法权益

用户应对自己的输入内容及生成结果自行负责。

开发者不对用户自行生成、保存、传播的内容承担责任。

# API 与数据说明

本项目部分功能依赖第三方 AI 服务。

用户使用时输入的 API Key（如有）仅保存在本地浏览器中，不会上传至开发者服务器。

游戏存档、设置及部分历史数据默认保存在浏览器本地缓存中。

开发者不保证存档永久有效，也不保证不同版本之间的存档兼容性。

请用户自行做好存档备份。

# 项目状态说明

本项目当前为实验性测试版本，内容可能随时修改，包括但不限于：

* 世界规则
* AI 行为逻辑
* 数值系统
* UI 与功能
* 存档结构

更新后可能出现：

* 存档失效
* 世界状态异常
* AI 输出变化
* 功能临时不可用

上述情况属于测试阶段正常现象。
